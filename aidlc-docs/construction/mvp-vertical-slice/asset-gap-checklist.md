# Asset Gap Checklist

List of required assets for the MVP and their legacy repo location.

- [ ] **UI Sprites & Panels**: Buttons, menus, modals.
  - *Location*: Legacy repo `assets/ui/` or similar.
- [ ] **Puzzle Assets**: 
  - Sudoku number tiles (1-6).
  - Memory match cards (front/back).
  - *Location*: Legacy repo `assets/puzzles/` or `assets/sprites/`.
- [ ] **Pet Assets**:
  - Minimum 1-2 Pet graphical assets (idle, basic animations if sprite sheets exist).
  - *Location*: Legacy repo `assets/pets/`.
- [ ] **Currency/Reward Icons**: Gold/Coins/Gems.
  - *Location*: Legacy repo `assets/icons/`.
- [ ] **Audio (Optional but recommended)**:
  - Button clicks, puzzle success chime, pet unlock sound.
  - *Location*: Legacy repo `assets/audio/`.

*Note: If any asset is missing from the legacy Godot repo, it will need to be identified during Unit U1 and flagged in the Risk Register.*
